Web Development Project 1 - Hispanic Eats
Submitted by: Duban Correa

This web app: A resource gallery featuring 15 unique Hispanic restaurants and food resources to help users discover authentic community eats.

Time spent: 12 hours spent in total

Required Features
The following required functionality is completed:

[x] The app has a cohesive, unique theme for events or resources relevant to a specific community

[x] Header/title describing the theme (Hispanic Eats) is displayed

[x] At least 10 unique events or resources are displayed in a responsive card format

[x] There are 15 cards displayed

[x] The cards are displayed in an organized grid format

[x] Each card includes information about the specific restaurant or resource

The following optional features are implemented:

[x] Buttons or links to a related resources are on each card component

[x] All cards have "View Menu" or "Visit Site" buttons

[x] The site is responsive for both desktop and mobile formats

[x] Web app is shown in a mobile format in the walkthrough
